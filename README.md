## Headspace_Wizards-1

# Libraries used
- Sympy for calculus logic
- Matplotlib and Streamlit plus charts for visualization

# Getting logic up and running
Use the "pip install sympy" in command prompt to install Sympy after you verify Sympy
Use the "pip install matplotlib" in command prompt to install Matplotlib.
Use the "pip install streamlit" in command prompt to install Streamlit.
Use the "pip install streamlit[charts]" in command prompt to install Streamlit Charts.
